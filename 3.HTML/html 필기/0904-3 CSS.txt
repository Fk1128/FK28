HTML 데이터셋 속성
커스템 사용자 속성을 DOM요소에 저장하는데 표준화된 방법.
HTML에서의 변수 역할이다 .

데이터 셋 사용의 장점

고유한 커스텀 값을 지정해 사용할 수 있다 .

데이터 셋 사용법
태그 내에 DATA로 시작하는 키워드를 기재하고 , 그 뒤에 하이픈(-) 이 조합된 형태로 
개발자가 정의하고싶은 속성명을 기재해주고 속성값을 써주면 사용자 변수가 완성된다 .

사용자 지정 데이터 속성(Custom data attributes)은
페이지 또는 응용프로그램에 대한 사용자 지정 '데이터', '상태,' '주석' 등을 저장하기 위한 것으로,
더 이상 적절한 표준 속성이나 요소가 없을 때 사용하는 HTML의 표준화된 속성입니다.
이 속성은 전역(글로벌) 속성 중 하나로, 모든 요소에 적용이 가능합니다.

주로 JS , CSS 에서 이 속성을 활용하여 동적인 기능추가 , 스타일 지정에 사용이된다 


<button data-id="341">좋아요</button>

데이터셋에 배열 , 객체 데이터저장
<div data-person='{"name":"Chris Coyier","job": "web Person"}'></div>

<div data-fruits='["apple","banana","melon",]></div>

CSS
- CSS는 HTML의 각 요소의 스타일을 정의하여 화면등에 어떻게 렌더링하면 되는지 브라우저에게 설명하기 위한 언어 
-Cascading Style Sheet의 약자
- 색상 , 폰트 , 각 요소의 위치변경 , 백그라운드 및 간단한 애니메이션 효과를 줄 수 있다 .

CSS 선택자
1. 태그의 이름으로 선택

2. 클래스(class) 선택자
- class속성: 특정 집단의 여러 요소를 한번에 선택할 때 사용

<h2 class="headings"> 이 부분에 스타일을 적용합니다 .</h2>
<h3 class="headings"> 이 부분에 스타일을 적용합니다 .</h3>
<h4 class="headings"> 이 부분에 스타일을 적용합니다 .</h4>


3. 아이디 선택자
- id속성 : CSS를 적용할 특정요소를 선택할 때 사용
- 문서당 ID값은 한번만 사용
-( # )은 CSS에서 id를 가리킨다

#a2{color : blue;}

색상을 표현하는 방법
CSS에서는 배경색, 글자색, 테두리의 색 등과 같이 색상을 다루는 경우가 자주있다 .

inherit 	
영단어로 상속이라는 의미이다 .
스타일 값으로 inherit을 지정을 하면 부모태그의 색상을 그대로 물려받는다 .
CSS의 기본값이며 따로 COLOR 속성을 지정하지 않는다면 자동으로 inherit로 지정된다 .

컬러네임 color name 
red,green,blue 등 색상을 의미하는 단어

RGB 또는 RGBA코드 (Decimal)
- RGB코드는 색상을 표현하는 갖아 단순한 코드이다.
(255,255,255)와 같이 3개의 숫자가 묶여있는 순서쌍으로 정의된다 .
각각 빛의 삼원색은 빨강, 초록 ,파란색을 의미한다 .

RGBA 는 투명도를 포함한다.
색상을 의미하는 RGB 뒤에 투명도를 의미하는 알파값이 추가된 형태

16진수 컬러코드(HEX RGB) 
색상을 16진수로 표현하는 것으로 컴퓨터와 디스플레이의 발달로 사용도가 매우 높아지고있다.

HTML 의 본질은 정보전달이며 , 정보전달의 핵심은 텍스트이다.
따라서 CSS에서는 텍스트를 아름답게 표현하기 위한 온갖 도구들이 준비되어있다 .

여백 지정하기
CSS에서 여백은 외부여백과 내부여백 두가지가 있습니다 .

1.외부여백(margin)
외부여백은 요소 바깥쪽의 여백을 의미 , 여백에는 다른요소가 침범할 수 없다 .
자기자신이 온전히 점유하는 고유영역이 생기는것이다 .


2. 내부여백 (padding)
요소 안쪽의 여백을 의미한다 .
